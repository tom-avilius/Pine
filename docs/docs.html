Documentation

  1. Complete video installation instructions available at: this youtube video.
  2. Install the splash application.
  3. Download the Pine theme.
  4. Move the theme folder (generally with the same name as that of the theme) into any directory or location where it would be safe and not be deleted. I recommend you to move it inside the documents folder under the splash/themes folder.
  5. Now with the splash application running go to the system tray (where audio and battery icons are in the taskbar).
  6. There you will find the splash icon, right click on it would reveal a menu.
  7. From the menu, choose the 'Load File' option.
  8. The file explorer would appear, find the theme folder you moved/extracted and find a file with the same name as that of the theme itself and ends with '.html' extension. Example: pine.html.
  9. Select the 'Open' option at the bottom of the file explorer.
  10. The selected file would be rendered. Check your homescreen.
  11. Now refresh the splash window by going to the homescreen. If splash is not running, launch it through the start menu.
  12. If you encounter any issue during the process, feel free to raise an issue on github or use the contact me page.
